# Shane's Pickleball Notebook V2

A mobile-first GitHub Pages notebook for pickleball cues, match notes, video links, and equipment notes.

## Files

- `index.html` controls the page structure.
- `styles.css` controls the design.
- `script.js` controls saving, editing, email, and text actions.
- `icon.svg` is the app icon.
- `manifest.json` lets the site behave more like an installed web app.

## How to update GitHub Pages

1. Open your Pickleball-Notebook GitHub repository.
2. Upload or replace these files in the main folder.
3. Commit the changes.
4. Refresh your GitHub Pages site.

Notes save automatically in the browser using localStorage.
